# landing-page/ specs

## index.html
LeakAid の需要検証用ランディングページ。単一HTMLファイルで完結。

- **Nav**: 固定ヘッダー。ロゴ（Inter フォント、モダンなサンセリフ）+ 「先行登録受付中」バッジ
- **Hero**: キャッチコピー「あなたの画像を、あなたの意思で取り下げる。」+ CTA ボタン
- **Trust Signals**: 4カード（無料相談・個人情報管理・法的知見に基づく対応・24時間以内に対応）
  - アイコンは SVG 線画（吹き出し・南京錠・盾・時計）、角丸背景つき
  - 全てAIベースのサービスであることを反映した文言
- **3 Steps**: URL共有 → AIが自動で削除申請 → 進捗をリアルタイム追跡
  - 「スタッフ」等の人的対応を示す表現は使用しない（全てAI自動処理）
  - 「削除を保証」と読めないようトーンを調整
- **CTA Form**: Formspree 連携（`YOUR_ID` はプレースホルダー）。メール登録フォーム
- **FAQ**: 3項目（費用・海外対応・プライバシー）。`<details>` 要素でアコーディオン実装
  - プライバシー回答でもAI自動処理である点を明記
- **Footer**: コピーライト + プライバシーポリシー・特商法リンク（アンカーのみ）+ 免責事項

### 技術仕様
- 外部依存: Google Fonts（Inter・Noto Sans JP・Noto Serif JP）
- CSS: インラインスタイル、レスポンシブ対応（600px ブレークポイント）
- JS: なし
- Vercel デプロイ対応（vercel.json 同梱）

## vercel.json
SPA 風のリライト設定。すべてのパスを index.html にルーティング。
